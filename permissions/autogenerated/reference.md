## Default Permission

This permission set configures the types of dialogs
available from the dialog plugin.

#### Granted Permissions

All dialog types are enabled.

#### This default permission set includes the following:

- `allow-ask`
- `allow-confirm`
- `allow-message`
- `allow-save`
- `allow-open`
